## IV. Navigation Structure (Revised Order)

The Sidebar menu order must be strictly as follows:

1.  **Dashboard** (Home)
2.  **Sales** (Bán Hàng)
3.  **Service** (Dịch Vụ)
4.  **Accounting** (Kế Toán)
5.  **Master Data** (Dữ Liệu Nguồn)  <-- *Moved Here*
6.  **Insurance** (Bảo Hiểm)
7.  **Admin** (Quản Trị)

*Rationale*: Master Data flows logically after Accounting (which sets prices) and before specialized modules like Insurance. Admin remains at the bottom as system config.
